Welcome to Microbe's documentation
==================================

**Microbe** is a `KISS`_ blog engine inspired by `Pelican`_ and powered by `Flask`_.

Write your contents using the `Markdown`_ syntax with a nice editor without needing to install a database. Pages are stored in flat files on the disk.

Administrate your blog directly from your browser.
