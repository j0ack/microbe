Welcome to Microbe's documentation!
===================================

Microbe is a Micro Blog Engine written in Python, inspired by `Pelican <http://docs.getpelican.com/>`_ and powered by Flask.

It has been developed following the `Kiss <https://en.wikipedia.org/wiki/Keep_it_Simple,_Stupid>`_ principles. No data base is required for the application. 

Write your contents using `Markdown <http://daringfireball.net/projects/markdown/>`_ format.

Installation
------------

    $ pip install microbe

Features
--------

- Blog posts and static pages
- Comments
- Theming support
- Atom feeds
- Code syntax hilighting
- Search
- Multi users
- Files upload
- Multi language support (English and French)

Technologies used
-----------------

- `Python <http://www.python.org/>`_ 
- The `Flask <http://flask.pocoo.org>`_ Python Web Framework
- `Flask-Flatpages <http://pythonhosted.org/Flask-FlatPages/>`_ extension
- `Flask-CodeMirror <https://pypi.python.org/pypi/flask-codemirror/0.0.2>`_ extension
- `Flask-Paginate <http://pythonhosted.org//Flask-paginate/>`_ extension
- `Flask-Themes2 <http://flask-themes2.readthedocs.org/en/latest/>`_ extension
- `Flask-WTF <https://flask-wtf.readthedocs.org/en/latest/>`_ extension
- `Pagedown Converter <https://code.google.com/p/pagedown/>`_
- `Whoosh <http://pythonhosted.org//Whoosh/>`_
- Markdown editor inspired from excellent `lepture editor <https://github.com/lepture/editor>`_
- `jQuery <http://jquery.com/>`_
- `GUnicorn <http://gunicorn.org/>`_
- `VizHash.js <https://github.com/sametmax/VizHash.js>`_
- `Foundation <http://foundation.zurb.com/>`_ CSS framework
